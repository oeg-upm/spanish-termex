Una parte clave de este problema es que un inspector sólo tiene acceso a los datos de una pequeña zona inspeccionada. En esta zona sólo existe un espesor mínimo, lo que no proporciona información suficiente para construir un modelo de los espesores más pequeños. Un inspector puede generar una muestra de las mediciones de espesor más pequeñas dividiendo los datos de inspección en varios bloques del mismo tamaño. En cada bloque se registra el espesor mínimo. Este conjunto forma una muestra de las medidas de espesor más pequeñas. A partir de esta muestra se puede construir un modelo que tenga en cuenta las variaciones de las medidas de espesor más pequeñas. El análisis de valores extremos (EVA) proporciona una forma limitante para este modelo. Afirma que, si las mediciones de espesor subyacente en cada bloque se toman a partir de distribuciones independientes e idénticas, entonces la muestra de mediciones de espesor mínimo seguirá una distribución de valores extremos generalizada (GEVD).
