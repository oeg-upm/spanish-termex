En los cálculos para la energía de formación, el tamaño de la caja se establece en 30a0×30a0×30a0, donde a0 es el parámetro de red bcc del Fe. Para todos los cálculos se utilizan condiciones de contorno periódicas y volumen constante. El algoritmo de Monte Carlo utilizado para determinar la configuración de energía más baja del grupo [28] se organiza de la siguiente manera. Primero, se investiga la energética de los vacíos sin helio. Se introduce una vacante en la celda de simulación y el sistema se minimiza utilizando un algoritmo de gradiente conjugado, dando como resultado una energía de formación de vacante única Evac de 1.72eV. A continuación, se elimina el átomo con la mayor energía potencial del sistema y nuevamente se minimiza el sistema. Este esquema se continúa de forma iterativa para crear vacíos hasta el número de vacantes objetivo y se calcula la energía de formación de cada uno. Luego, se introducen átomos de helio en las vacantes. Se mide y registra la energía total del sistema. En este punto, se utiliza un esquema MC de Metropolis [29] para encontrar las configuraciones de baja energía. Cada átomo de helio en el sistema se desplaza aleatoriamente desde su posición hasta un máximo de rmax (4.5Å, la distancia de corte para las interacciones He-He) en cada una de las direcciones x, y y z y luego se minimiza utilizando el algoritmo de gradiente conjugado. Cada burbuja se continúa durante un mínimo de 10,000 pasos. Después de eso, las búsquedas se terminarán si la energía del sistema no disminuye en 10 pasos adicionales.
