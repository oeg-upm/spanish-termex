La selección del contratista para un proyecto es una decisión importante, una para el tiempo y el costo del proyecto, luego para la calidad obtenida por el proyecto. Aunque los gerentes de proyecto pueden determinar fácilmente el tiempo y el costo del proyecto, la calidad suele ser indefinida, especialmente para gerentes sin experiencia. Con una propiedad aprendible, se introduce primero en este documento un enfoque para cuantificar la calidad obtenida para un proyecto de perforación de pozos de gas. Luego, basándose en estos tres objetivos (tiempo, costo y calidad), un problema de selección de contratista se convierte en un problema de optimización. A continuación, se utiliza el algoritmo NSGA-II para la solución. Al final, se realiza un análisis de sensibilidad para seleccionar los parámetros del algoritmo.
