Se propone un modelo para modelar servicios web centrados en datos que están alimentados por bases de datos relacionales e interactúan con usuarios de acuerdo con fórmulas lógicas que especifican restricciones de entrada, restricciones de flujo de control y reglas de estado/salida/acción. Las fórmulas de Lógica Lineal Temporal de Primer Orden (LTL-FO) sobre entradas, estados, salidas y acciones se utilizan para expresar las propiedades a verificar. Hemos demostrado que la verificación automática de propiedades LTL-FO de servicios web centrados en datos bajo restricciones de entrada acotadas es decidible al reducir los servicios web a aplicaciones web centradas en datos. Por lo tanto, podemos verificar las especificaciones de servicios web utilizando verificadores existentes diseñados para aplicaciones web.
