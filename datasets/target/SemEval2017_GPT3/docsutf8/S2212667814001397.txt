En este documento, se propone un método basado en análisis de regresión para calcular el Puntaje de Influencia de Revistas. Este Puntaje de Influencia se utiliza para medir la influencia científica de revistas académicas. El Puntaje de Influencia de Revistas se calcula utilizando varios factores de manera ponderada. Luego, se compara el Puntaje con el Puntaje de Revistas SCImago. Los resultados muestran que el error es pequeño entre los métodos existentes y propuestos, demostrando que el modelo es una forma factible y efectiva de calcular el impacto científico de las revistas.
