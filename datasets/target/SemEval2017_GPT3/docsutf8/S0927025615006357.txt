En este artículo, se ha aplicado un modelo de plasticidad cristalina, en combinación con XFEM, para estudiar la deformación cíclica y el crecimiento de grietas por fatiga en una superaleación a base de níquel LSHR (Low Solvus High Refractory) a alta temperatura. El primer objetivo de esta investigación fue desarrollar y evaluar un modelo de elemento finito basado en RVE con la incorporación de una microestructura material realista. El segundo objetivo de este trabajo fue determinar los parámetros de un modelo constitutivo de plasticidad cristalina para describir el comportamiento de deformación cíclica del material mediante el uso de un subrutina de material definida por el usuario (UMAT) interfazada con el paquete de elementos finitos ABAQUS. Los parámetros del modelo se calibraron a partir de extensos análisis de elementos finitos para ajustarse a los datos de ensayos monotónicos, de relajación de esfuerzos y cíclicos. El tercer objetivo fue predecir el crecimiento de grietas combinando la técnica XFEM y el UMAT de plasticidad cristalina calibrado, para lo cual se utilizó la deformación plástica acumulada como criterio de fractura.
