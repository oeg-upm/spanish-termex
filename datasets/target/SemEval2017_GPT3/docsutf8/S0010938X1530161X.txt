Una parte clave de este problema es que un inspector solo tiene acceso a datos de una pequeña área inspeccionada. En esta área, solo hay un grosor mínimo, que no proporciona suficiente información para construir un modelo de los grosores más pequeños. Un inspector puede generar una muestra de las mediciones de grosor más pequeñas dividiendo los datos de inspección en varios bloques de tamaño igual. En cada bloque se registra el grosor mínimo. Este conjunto forma una muestra de las mediciones de grosor más pequeñas. A partir de esta muestra, se puede construir un modelo que tenga en cuenta las variaciones de las mediciones de grosor más pequeñas. El análisis de valores extremos (EVA) proporciona una forma límite para este modelo. Indica que, si las mediciones de grosor subyacentes en cada bloque se toman de distribuciones independientes e idénticas, entonces la muestra de las mediciones de grosor mínimo seguirá una distribución generalizada de valores extremos (GEVD).
