Se propone un método de control predictivo de modelo fuzzy-Hammerstein para un reactor continuo de tanque agitado (CSTR). En este artículo se utiliza un modelo fuzzy T-S para aproximar las características no lineales estáticas del modelo Hammerstein, y se utiliza un modelo autorregresivo lineal para resolver los resultados del control óptimo. El controlador predictivo no lineal diseñado utilizando el modelo Hammerstein aprovecha la capacidad de enfoque universal no lineal del modelo T-S, y divide la cuestión del control predictivo no lineal en el reconocimiento del modelo no lineal y la cuestión del control predictivo lineal. Los resultados de aplicación del proceso CSTR muestran que el método de control propuesto tiene un buen rendimiento de control en comparación con el controlador PID.
