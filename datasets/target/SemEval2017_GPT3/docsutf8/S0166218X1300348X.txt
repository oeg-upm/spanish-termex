Los programas max-lineales se han utilizado para describir problemas de optimización para sistemas interactivos multiprocesador. En algunos casos, las variables utilizadas en este modelo deben ser enteras; sin embargo, no parece existir un método para encontrar soluciones enteras a los programas max-lineales. Para una clase genérica de matrices, demostramos que las soluciones enteras a sistemas y programas max-lineales de dos lados pueden encontrarse en tiempo polinómico. Para matrices generales, adaptamos los métodos existentes para encontrar soluciones reales con el fin de obtener algoritmos para encontrar soluciones enteras.
