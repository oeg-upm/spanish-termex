Un número de parámetros del modelo pueden cambiar regional o estacionalmente, en particular las propiedades ópticas inherentes de los constituyentes del agua [ai⁎(λ), aY⁎(λ), aD⁎(λ), bX(λ), bb,X⁎, bb,Mie⁎] y las propiedades ópticas aparentes del fondo [Rib(λ), Bi] y la atmósfera. La base de datos proporcionada con WASI se ha derivado de mediciones in situ de lagos en el sur de Alemania (Gege, 1998; Heege, 2000; Pinnel, 2007). Si no se dispone de información específica del sitio, también se puede utilizar como una primera aproximación para otros ecosistemas. La variabilidad dentro de un ecosistema puede ser tan grande como entre diferentes ecosistemas, es decir, conjuntos de propiedades ópticas específicas del ecosistema no existen. Sin embargo, se debe utilizar la información específica de la región o la temporada siempre que esté disponible. Idealmente, las propiedades ópticas deberían medirse en el sitio de prueba cerca del sobrevuelo del avión o satélite. Sin embargo, esto no siempre es posible. Una fuente valiosa de información es la página web de IOCCG (IOCCG, 2013b). Mantiene una lista de enlaces a conjuntos de datos disponibles públicamente, como por ejemplo la base de datos IOCCG (2006), el conjunto de datos del Algoritmo Marino Bio-Óptico de la NASA (NOMAD) y el Sistema de Archivo y Almacenamiento Bio-Óptico SeaWiFS (SeaBASS).
