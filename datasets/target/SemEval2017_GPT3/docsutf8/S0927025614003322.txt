Un principio de la ciencia de materiales de alto rendimiento es que uno no sabe de antemano dónde se encuentra el valor de los datos para una aplicación específica. Las tendencias y percepciones se deducen a posteriori. Esto requiere interfaces eficientes para interrogar datos disponibles en varios niveles. Hemos desarrollado una sencilla API basada en la WEB para mejorar en gran medida la accesibilidad y utilidad de la base de datos AFLOWLIB [14] para la comunidad científica. A través de ella, el cliente puede acceder a propiedades físicas calculadas (termodinámicas, cristalográficas o mecánicas), así como a la procedencia de la simulación y propiedades de tiempo de ejecución de los sistemas incluidos. Los datos pueden ser utilizados directamente (por ejemplo, para explorar una clase de materiales con una propiedad deseada) o integrados en flujos de trabajo de nivel superior. La interfaz también permite compartir actualizaciones de datos utilizados en trabajos previamente publicados, por ejemplo, diagramas de fases de aleaciones previamente calculados [19–31], por lo tanto, la base de datos puede expandirse sistemáticamente.
