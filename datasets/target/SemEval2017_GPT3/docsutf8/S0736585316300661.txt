Para calcular índices de precios hedónicos en el modelo lineal, es necesario calcular el precio inicial o de referencia (Triplett, 2006). El presente estudio adopta el enfoque de de Haan y Diewert (2013): se construye un índice de precios utilizando el precio generado por los coeficientes estimados de un modelo de regresión del período base, y se calcula en base a los valores promedio del período base de una característica dada del plan de celular z¯ para cada operador (Tabla Suplementaria S5). Para características continuas, se utilizan promedios directos; para características binarias, se utilizan las proporciones de planes de celular que contienen la característica. Los precios resultantes para este plan de celular promedio se convierten a un índice aplicando los cambios de precio puro calculados previamente (δs). Finalmente, el índice de precios hedónicos general se calcula como el promedio ponderado de los índices a nivel de empresa. Las ponderaciones corresponden a la proporción relativa de planes de celular por operador en la muestra (0.3534 para HT, 0.3212 para Vip y 0.3254 para Tele2).
