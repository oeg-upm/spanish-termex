La selección de contratistas para un proyecto es una decisión importante, una para el tiempo y el costo del proyecto, la siguiente para la calidad obtenida por el proyecto.Aunque los gerentes de proyecto pueden determinar fácilmente el tiempo y el costo del proyecto, la calidad generalmente está indefinida, especialmente para los gerentes no experimentados.Con una propiedad aprendible, se introduce primero un enfoque en este documento para cuantificar la calidad obtenida para un proyecto de perforación de pozos de gas.Luego, según estos tres objetivos (tiempo, costo y calidad), un problema de selección de contratistas se convierte en un problema de optimización.A continuación, el algoritmo NSGA-II se utiliza para la solución.Al final, se realiza un análisis de sensibilidad para seleccionar los parámetros del algoritmo.
