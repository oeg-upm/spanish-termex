En este documento, el modelo de plasticidad de cristal, en combinación con XFEM, se ha aplicado para estudiar la deformación cíclica y el crecimiento de la grieta de fatiga en un Superalloy LSHR a base de níquel (bajo solvus alto refractario) a alta temperatura.El primer objetivo de esta investigación fue desarrollar y evaluar un modelo de elementos finitos basado en RVE con la incorporación de una microestructura de material realista.El segundo objetivo de este trabajo fue determinar los parámetros de un modelo constitutivo de plasticidad cristalina para describir el comportamiento de deformación cíclica del material mediante el uso de una subrutina de material definida por el usuario (UMAT) interactuada con el paquete de elementos finitos ABAQUS.Los parámetros del modelo se calibraron a partir de análisis extensos de elementos finitos para adaptarse a los datos monotónicos, relajación del estrés y de prueba cíclica.El tercer objetivo era predecir el crecimiento de grietas combinando la técnica XFEM y la plasticidad de cristal calibrada UMAT, para la cual se usó la tensión plástica acumulada como criterio de fractura.
