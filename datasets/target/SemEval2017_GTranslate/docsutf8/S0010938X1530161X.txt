Una parte clave de este problema es que un inspector solo tiene acceso a datos de un área inspeccionada pequeña.En esta área, solo hay un grosor mínimo, que no proporciona suficiente información para construir un modelo de los espesores más pequeños.Un inspector puede generar una muestra de las mediciones de espesor más pequeñas al dividir los datos de inspección en varios bloques de tamaño igualmente.En cada bloque se registra el grosor mínimo.Este conjunto forma una muestra de las mediciones de espesor más pequeñas.A partir de esta muestra, se puede construir un modelo que tenga en cuenta las variaciones de las mediciones de espesor más pequeñas.El análisis de valor extremo (EVA) proporciona una forma limitante para este modelo.Establece que, si las mediciones de grosor subyacente en cada bloque se toman de distribuciones independientes e idénticas, entonces la muestra de mediciones de espesor mínimo seguirá una distribución generalizada de valor extremo (GEVD).
