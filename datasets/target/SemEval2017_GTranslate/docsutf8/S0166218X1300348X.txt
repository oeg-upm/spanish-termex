Los programas máximos lineales se han utilizado para describir los problemas de optimización para los sistemas interactivos multiprocesador.En algunos casos, las variables utilizadas en este modelo deben ser enteras;Sin embargo, no parece existir ningún método para encontrar soluciones enteras para programas máximos lineales. Para una clase genérica de matrices, mostramos que las soluciones enteras a los sistemas y programas máximos de dos lados se pueden encontrar en el tiempo polinómico.Para las matrices generales, adaptamos los métodos existentes para encontrar soluciones reales para obtener algoritmos para encontrar soluciones enteras.
