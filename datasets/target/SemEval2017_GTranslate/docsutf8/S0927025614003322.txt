Un principio de la ciencia de los materiales de alto rendimiento es que uno no conoce a priori donde el valor de los datos se encuentra para cualquier aplicación específica.Las tendencias y las ideas se deducen a posteriori.Esto requiere interfaces eficientes para interrogar los datos disponibles en varios niveles.Hemos desarrollado una API simple basada en la web para mejorar en gran medida la accesibilidad y la utilidad de la base de datos Aflowlib [14] a la comunidad científica.A través de él, el cliente puede acceder a propiedades físicas calculadas (propiedades termodinámicas, cristalográficas o mecánicas), así como la procedencia de simulación y las propiedades de tiempo de ejecución de los sistemas incluidos.Los datos pueden usarse directamente (por ejemplo, para explorar una clase de materiales con una propiedad deseada) o integrarse en flujos de trabajo de nivel superior.La interfaz también permite compartir actualizaciones de datos utilizados en trabajos publicados anteriores, por ejemplo, diagramas de fase de aleación previamente calculados [19–31], por lo que la base de datos se puede ampliar sistemáticamente.
