El beneficio principal de usar un modelo 3D es que permite la aplicación de propiedades del material anisotrópico.Como una estructura de celosía llena hexagonal, un solo grano de circonio es plásticamente anisotrópico debido a la dificultad de activar el deslizamiento con un componente 〈c〉 [23–26].Abaqus permite que esto se represente estableciendo las relaciones potenciales de plasticidad.Las constantes elásticas y de plástico anisotrópicas se muestran en la Tabla 1. Las aleaciones de circonio a menudo pueden tener una distribución de postes basales bimodales, con una inclinación en la dirección normal o C basal de ± 30 ° en la dirección normal que se cita para circaleo recristalizado-4 [27 [27, 28].Sin embargo, para simplificar la dirección normal o C basal se ha tomado como paralelo a la dirección normal.Como tal, las direcciones 1, 2 y 3 en la Tabla 1 se correlacionan con el sistema de coordenadas global X, Y y Z para las simulaciones 3D, con la dirección 3 correlacionada con la dirección C de una red de unidades de circonio.La Tabla 1 también muestra las propiedades elásticas incorporadas en las simulaciones.La capa de óxido se ha simulado como material puramente elástico.Aunque se sabe que el óxido está fuertemente texturizado [29], todavía se simula como un sólido homogéneo, por lo tanto, las propiedades del material isotrópico se han utilizado para el óxido en todas las simulaciones.
