Los métodos de desgarro e interconexión de elementos finitos (FETI) son un enfoque poderoso para diseñar solucionadores para problemas a gran escala en la mecánica computacional.El problema de simulación numérica se subdivide en una serie de subproblemas independientes, que luego se acoplan de manera apropiada.El análisis isogeométrico (IgA) basado en B-Spline (IgA) basado en B NURBS (no uniforme) aplicado a geometrías complejas requiere representar el dominio computacional como una colección de varias geometrías NURBS.Dado que existe una descomposición natural del dominio computacional en varios subdominios, la IGA basada en Nurbs es particularmente adecuada para usar métodos FETI. Este artículo propone el nuevo método de desgarro y interconexión de interconexión (IETI), que combina el diseño de solucionador avanzado de FETI con FETI con FETI con FETI con FETI con FETI con FETI con FETI conLa representación exacta de geometría de IgA.Describimos el marco IETI para dos clases de problemas de modelo simple (Poisson y elasticidad linealizada) y discutimos el acoplamiento de los subdominios a lo largo de las interfaces (tanto para interfaces coincidentes como para interfaces con juntas T, es decir, nodos colgantes).Se presta especial atención a la construcción de un preacondicionador adecuado para el solucionador lineal iterativo utilizado para el problema de la interfaz.Reportamos varios experimentos computacionales para demostrar el rendimiento del método IETI propuesto.
