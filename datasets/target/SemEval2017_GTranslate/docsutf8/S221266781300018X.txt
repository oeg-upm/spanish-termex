Se proponen Amodel para modelar servicios web centrados en datos que funcionan con bases de datos relacionales e interactúan con los usuarios de acuerdo con fórmulas lógicas que especifican restricciones de entrada, restricciones de flujo de control y reglas de estado/salida/acción.Las fórmulas lineales de lógica de primer orden temporal (LTL-FO) sobre entradas, estados, salidas y acciones se utilizan para expresar las propiedades que se verificarán. Hemos demostrado que la verificación automática de las propiedades LTL-FO de los servicios web centrados en los datos bajo la entrada de la entrada-Las restricciones anuladas son decidibles al reducir los servicios web a aplicaciones web centradas en datos.Por lo tanto, podemos verificar las especificaciones del servicio web utilizando el verificador existente diseñado para aplicaciones web.
