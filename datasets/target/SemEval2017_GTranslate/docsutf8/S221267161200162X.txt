Se propone un método de control predictivo del modelo difuso-hammerstein para un reactor continuo de tanque agitado (CSTR).En este documento, el modelo difuso T-S se utiliza para aproximar las características no lineales estáticas del modelo Hammerstein, y se utiliza un modelo autorregresivo lineal para resolver los resultados del control óptimo.El controlador predictivo no lineal diseñado que utiliza el modelo Hammerstein hace un buen uso de la capacidad del enfoque universal no lineal del modelo T-S, y divide la cuestión del control predictivo no lineal en la recongnización del modelo no lineal y la cuestión del control predictivo lineal.Los resultados de la aplicación del proceso CSTR muestran que el método de control propuesto tiene un buen rendimiento de control en comparación con el controlador PID.
