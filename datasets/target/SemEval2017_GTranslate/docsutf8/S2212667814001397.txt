En este documento, se propone un método basado en el análisis de regresión para calcular el puntaje de influencia del revista.Este puntaje de influencia se utiliza para medir la influencia científica de las revistas académicas.El puntaje de influencia del diario se calcula utilizando varios factores de manera ponderada.El puntaje se compara con la puntuación de Scimago Journal.Los resultados muestran que el error es pequeño entre los métodos existentes y propuestos, lo que demuestra que el modelo es una forma factible y efectiva de calcular el impacto científico de las revistas.
