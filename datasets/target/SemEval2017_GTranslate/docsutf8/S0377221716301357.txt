Como se mencionó anteriormente, este documento representa esfuerzos continuos para abordar eficientemente el MPSP estocástico.El trabajo futuro puede considerar investigar si el algoritmo sería tan exitoso o no para resolver variantes del MPSP que incluyen más restricciones operativas, como el grado de corte variable, la mezcla de grado y el almacenamiento, como lo es para resolver la variante clásica considerada eneste papel.De hecho, es un algoritmo de uso general y debe ser aplicable a cualquiera de estas variantes.Otras vías de investigación incluyen considerar otras estrategias para actualizar las sanciones dentro del pH y otros métodos para resolver los subproblemas.Finalmente, otra dirección de investigación importante es el desarrollo de otros enfoques de solución eficientes.Dado que se ha observado empíricamente que la formulación del problema a menudo logra pequeñas brechas de integralidad, un enfoque podría ser resolver la relajación lineal del problema utilizando un algoritmo eficiente y luego usar un procedimiento de reducción de LP para obtener una solución entera.
